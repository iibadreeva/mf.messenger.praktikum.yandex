## Описание
- Статику раздает `express` на `nodejs`;
- Защита от DOS атак`express-rate-limit`;
- Верстка `HTML`, `CSS`, `БЭМ`;
- Проэкт написан с использование `SCSS` `TYPESCRIPT`
- Реализация `fetch`;
- Тесты написаны на `Jest`;

### Для использования
```
npm install -g sass
```

# Запуск проэкта
```sh
npm run start
```

# Запуск стилей
```sh
npm run style
```

# Запуск теста
```sh
npm run test
```
